# **App Name**: SynergyX Portfolio

## Core Features:

- Landing Page Showcase: Dynamic hero section highlighting key skills and projects.
- Marketing Case Studies: Dedicated section showcasing successful marketing campaigns with detailed analysis.
- Web3 Project Gallery: Interactive gallery featuring Web3 projects, integrated with blockchain data.
- Video Editing Reel: High-quality video reel showcasing video editing skills and projects.
- AI-Powered Content Tool: AI Tool assists the user in updating sections on the website by reasoning on what content to add, given a general description of the section that needs to be updated. The updates will follow a specific format (e.g. description followed by title of a project) which must be inferred by the tool.
- Contact Form: Interactive contact form with integrated form validation and submission handling.

## Style Guidelines:

- Primary color: Neon blue (#4AF4FF) for a futuristic and vibrant feel.
- Background color: Dark navy blue (#19293B) for a premium, matte aesthetic.
- Accent color: Electric purple (#BE52F2) for highlighting key elements.
- Body and headline font: 'Space Grotesk' sans-serif for a modern, tech-oriented style; 'Inter' for longer body text.
- Use minimalist, geometric icons with a neon glow effect.
- Futuristic grid-based layout with smooth transitions and parallax scrolling effects.
- Incorporate motion design elements such as subtle animations, micro-interactions, and loading animations.
